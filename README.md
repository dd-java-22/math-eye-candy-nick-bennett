# Math Eye Candy

## Summary

This is an app that allows the user to create interesting images&mdash;and learn about some generative and algorithmic art concepts&mdash;by executing one of a handful of algorithms. For the initial version, we anticipate providing support for drawing these image types:

- Chaos game
- Langton's ant
- Hypotrochoids and epitrochoids
- K-means pointillism
- Traveling salesman problem (TSP)
